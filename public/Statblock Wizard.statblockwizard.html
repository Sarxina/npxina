Traits

NPXina Viewer.The Viewer is where you see or download the formatted stat block.

NPXina Creator.The Creator is the place where you create or edit the content of your stat block. You enter the text, NPXina handles the layout.

Actions

Viewer Actions.In the Viewer, these actions are available:

Creator.
Open the Creator.
Columns
Toggle between 1 and 2 column view. The selected option affects the JSON, HTML, and PNG file export.
Transparent.
Loop through normal, semi transparent, and fully transparent backgrounds. The selected option affects the HTML and PNG file export.
Upload JSON.
Load a NPXina json file.
JSON.
Download as a NPXina json file. This contains everything required to later view or edit the stat block again. The name of the stat block will be used for the name of the file.
HTML.
Download as a NPXina partial html file, containing a DIV element that you can use in your own html files. The file needs a style sheet!
CSS.
Get the style sheet that is used by NPXina. You are free to use, edit, or redistribute this CSS file. The referenced fonts are available on the internet under the SIL Open Font License.
PNG.
Download the NPXina PNG image file. Some browsers need the zoom level to be 100% to calculate the image size correctly.

Creator Actions.In the Creator, these actions are available:

New Stat Block.
Start a completely new stat block.
Upload JSON.
Load a NPXina json file.
Viewer.
Save the current stat block in the browser's local storage, then show it in the Viewer.
Bonus Actions

Home Logo.Select the logo in the top of the screen to get back to the app's startup page.

Reactions

NPXina's reactions only work if supported by your device and browser.

Hover.Trigger:You hover your mouse pointer over a button.Response:Extra information about this button's function is shown in a tooltip.

Shortcut Keys.You can use a keyboard to select many functions. The Hover feature shows the appropriate keys. Depending on your browser, you need to press a key combination, like Alt+key or Control+Option+key.

Drag & Drop.Trigger:You drop a NPXina json file on the current stat block in the Viewer.Response:The file is loaded and the stat block is shown.

Legendary Actions

Legendary Action Uses: 1. Immediately after opening the home page, NPXina can expend a use to take one of the following actions. NPXina regains all spended uses at the start of each of its turns.

Version Selection.Select the Original version or the 2024 version of the stat blocks. Your selection impacts the 'View' and 'Create or edit' Legendary actions.

View a stat block.Open the Viewer to see your current stat block.

Create or edit a stat block.Open the Creator to work on the stat block.

Legal information.Shows information about usage options and licenses. Send us a message using the email link at the top.

Clear saved stat block.Remove all data that was stored by this app from your browser's storage. If there is no stored stat block, the Creator and the Viewer will default to the 'Statblock Wizard' demo stat block.

Epic Actions

Image.NPXina supports one image in the stat block, which is embedded in the JSON and HTML files. It is also shown in the PNG files.

Note: Stat blocks of this style usually have no image.

Print.Use your browser's Print action from the Viewer to print the stat block using a printer-friendly styling with less color. The image is not printed. If you want to print a full-color version or to include the image, use the PNG file.
